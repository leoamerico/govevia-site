# EVID-CLM-PACKAGE-HASH — Manifest + hashes (lab-safe)

## Objetivo

Demonstrar integridade e reprodutibilidade do pacote gerado (manifest + SHA-256).

## Como reproduzir

```bash
npm run clm:build -- --account unai-mg
```

## Limites e Condições

- Pacote é proposta/minuta: contratação final depende do rito aplicável e aceite formal do órgão.
