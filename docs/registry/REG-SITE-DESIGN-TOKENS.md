# REG-SITE-DESIGN-TOKENS (cheatsheet)

## Princípios
- Usar somente classes/tokens já existentes no projeto (sem HEX, sem `@import`, sem fontes externas).
- Preferir utilitários existentes: `container-custom`, `btn-primary`, `btn-secondary`, tokens `text-institutional-*`, `bg-*` já aprovados.

## Exemplos (padrões aprovados)
- Tipografia: `font-serif`, `font-sans`, `text-institutional-graphite`, `text-institutional-navy`
- Layout: `container-custom`, `max-w-5xl`, `mx-auto`, `pt-32 pb-12`
- Superfícies: `bg-gradient-to-b from-institutional-offwhite to-white`, `border-primary/20`, `bg-primary/5`

## Componentes base (mínimo)
- Botão: `btn-primary`, `btn-secondary`
- Input: `rounded-md border border-gray-300 bg-white px-4 py-3 ...`
- Card: `rounded-md border ... p-4`
